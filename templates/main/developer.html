TFSToolset

WPF application for TFS query management

GitHub Repository More information...
Description

WPF application, built in Visual Studio 2013, utilizing the MahApps.Metro toolkit for Metro-style UI design. Allows the user to easily add query sub-folders to specified TFS projects, as well as copying queries from one sub-folder to another, making sprint and iteration query management seamless. Also allows the user to rename specific text within queries (i.e., "Sprint 14" to "Sprint 15") for more efficient sprint-to-sprint query usage.

Technologies utilized:
C#
Windows Presentation Foundation
XAML
